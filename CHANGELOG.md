# 변경 이력

이 프로젝트는 [Keep a Changelog](https://keepachangelog.com/ko/1.1.0/) 형식을 참고합니다.

## [0.2.0] - 2026-07-25

### 추가

- 채점 실행별 이력, 원본 답안지 보존, SQLite 기반 결과·진단 기록
- 이력 검색·상세 열람·원본 확인·백업 ZIP·복구 가능한 삭제
- 새 민감도로 재분석, 새 정답표로 재채점, 변경 전후 미리보기
- 학생별 수동 답안 보정과 재계산, 수정 사유 진단 기록
- 문항별 정답률·선택지 분포·점수 요약 분석
- 교수용 요약 작업공간과 증거 중심 결과 상세 화면
- JSONL 진단 로그와 비정상 종료된 작업의 다음 실행 시 감지
- 조명 편차 보정과 1~10 인식 민감도 조절을 통한 흐린 마킹 검토 지원

### 유지

- 기존 Windows 단독 실행 방식과 `채점결과.xlsx`, `debug/` 산출물
